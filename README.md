# 🎬 Utube - YouTube Clone

A simple YouTube-style video listing page using FreeAPI.

## 🚀 Features
✅ Fetch & display YouTube videos  
✅ Responsive grid layout  
✅ Search bar for filtering videos  
✅ Click to watch on YouTube
